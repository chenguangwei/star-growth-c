# 星星成长系统 ✨

一个专为孩子设计的奖励系统，采用现代化的苹果风格UI设计，帮助父母记录孩子的学习任务完成情况，孩子可以用获得的星星兑换奖励。

## 功能特点

- 🎯 **每日任务打卡** - 记录各种学习任务，自动计算星星奖励
- 📝 **单元测验记录** - 记录测验成绩，根据成绩自动计算奖励
- 🎁 **奖励兑换站** - 用星星兑换心仪奖励，支持钱币换算
- 📊 **数据统计** - 可视化查看学习数据和趋势分析
- 👥 **多孩子管理** - 支持管理多个孩子的独立数据
- 🔒 **父母管理后台** - 父母可调整数据、管理孩子档案
- 📱 **响应式设计** - 支持手机、平板、电脑访问

## 技术栈

- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS
- **UI组件**: shadcn/ui
- **动画**: Framer Motion
- **数据可视化**: Recharts
- **图标**: Lucide React
- **数据存储**: localStorage (开发阶段)


打开 [http://localhost:3000](http://localhost:3000) 查看应用。

### 构建生产版本

```bash

```

## 使用说明

### 首次使用

1. 启动应用后，系统会提示添加孩子
2. 在"孩子管理"页面添加第一个孩子的信息（姓名、头像）
3. 开始记录每日任务和测验成绩

### 每日任务

- 点击任务卡片标记完成
- 有次数限制的任务（如专注番茄奖）可以多次完成
- 需要输入的任务（如听写挑战）会弹出输入对话框
- 注意：每天低于3个星星将归0

### 单元测验

- 选择科目和成绩等级（3星/4星/5星）
- 系统自动计算奖励星星
- 进步会有额外奖励
- 3星成绩会触发"智慧星"任务

### 奖励兑换

- 在兑换站查看可用奖励
- 根据当前星星数显示不同的兑换比例
- 兑换后星星会相应扣除

### 父母管理后台

- 默认密码：`admin123`
- 可以调整星星数、标记欺骗行为
- 查看所有孩子的数据
- 管理孩子档案

## 奖励规则

### 每日任务（基础分）

- 准时启动奖：+1 ⭐️
- 晨读之星：+1 ⭐️
- 听写/默写挑战：根据正确率 +3~5 ⭐️
- 专注番茄奖：+2 ⭐️（每天最多3次）
- 抗干扰小勇士：+1 ⭐️（每天最多4次）
- 清单完成奖：每两项任务 +1 ⭐️
- 其他任务详见设置页面

### 单元测验

- 3星：触发智慧星任务 +5 ⭐️
- 4星：+6 ⭐️（3星到4星进步额外+8 ⭐️）
- 5星：+15 ⭐️

### 兑换规则

- 低于40星星：3个星星 = 1元
- 超过40星星：2个星星 = 1元
- 超过200星星：星星数 × 0.8 = 兑换钱币数

## 注意事项

- 数据存储在浏览器的localStorage中，清除浏览器数据会丢失所有记录
- 建议定期备份数据（导出功能待开发）
- 父母管理后台密码可以在代码中修改（默认：admin123）
- 所有操作都有操作日志记录（在管理后台可见）
